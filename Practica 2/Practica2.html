Bienvenido a mi Pagina HTML

HTML es el lenguaje con el que se escriben las paginas web. Las páginas web pueden ser vistas por el usuario mediante un tipo de aplicación llamada navegador.

Este lenguaje nos permite aglutimar textos, sonidos e imágenes y combinarlos a nuestro gusto.

el HTML nos permite la introducción de referencias a otras páginas por medio de los enlaces hipertextos.
